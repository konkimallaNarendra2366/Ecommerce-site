## 📌 Project Overview
The Online Shopping Cart E-Commerce Website is a web-based application developed using Java, JDBC, Servlets, JSP, HTML, CSS, JavaScript, and MySQL. It enables customers to browse products, add items to a shopping cart, manage quantities, and place orders seamlessly. The system provides a secure and user-friendly shopping experience while allowing administrators to manage products and customer data efficiently.


## ✨ Features

- User Registration and Login Authentication
- Browse Products by Categories
- Product Search Functionality
- Add Products to Shopping Cart
- Remove Items from Cart
- Increase or Decrease Product Quantity
- Automatic Price Calculation
- Secure Checkout Process
- Order Confirmation
- Responsive User Interface
- Session Management using Servlets
- Database Connectivity using JDBC
- Admin Panel for Product Management
- CRUD Operations for Products
